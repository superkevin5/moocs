This is an alpha prototype dashboard produced by the LTU LADA team in order to characterise the activity in MOOCs.
In order to facilitate the organisation of the large amount of data and details we provided three paths to explore the data.

  • On the left there is an overview by categories of information.
  • On the right the information is presented based on functional domains.
  • A full site maps with links to individual section is provided by expanding the pull menu on the right side of the page.

MOOC Dashboard Overview

This dashboard is a tool which aims to provide easily accessible, at-a-glance information to better understand different aspects from a MOOC course.

It draws from available data to produce snapshot visual representations. The dashboard is designed to initiate conversation, investigation and/or action around different course aspects.

 

Visual Walkthrough of the main user interface components:

Home page overview:

  • (a) Report Categories – data segmented by key areas
  • (b) Functional Domains – data segmented course elements
  • (c) Quick links & site map
  • (d) Help buttons section

 

Example page “Overview from Report Categories”:

  • (e) Navigation menu (to move between pages)
  • (f) Help – to get more information about a page
  • (g) Dashboard Guide – A quick guide to common features in the dashboard.
  • (h) Export tools – self-service tools to allow data download and further individual exploration

 

Tabs with a page:

  • (i) Tabs help separate larger content areas into manageable chunks.

 

 

Key Definitions:

  • Signature Track, refers to those who paid for a certificate
  • Non-Signature Track, refers to those who did not pay for the course
  • Distinction, refers to those who achieved a grade as per the formula for distinction
  • Pass, refers to those who achieved a grade as per the formula for distinction

 

 

Additional Notes:

  • Data is shown only for students (and excludes teacher/tutor/community mentor data)
  • Weeks 1..52 refer to calendar weeks, and situate when during the year, the course ran.
